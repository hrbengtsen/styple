### API Reference

See [Radix-UI AlertDialog (v0.1.7)](https://www.radix-ui.com/docs/primitives/components/alert-dialog) for details. This only covers **styple** abstractions.

#### AlertDialog

A styled component wrapper for `AlertDialog.Root`.

#### AlertDialogContent

A styled abstraction combining `AlertDialog.Overlay`, `AlertDialog.Portal` and `AlertDialog.Content`.

#### Others

The rest of the components of the primitive are exported by **styple** with no changes. Aria-labels can be used as an alternative to `AlertDialog.Title` and `AlertDialog.Description` as in the example above.
