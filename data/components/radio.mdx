### Example

<Preview>

    css={{

        value="1"

### API Reference

See [Radix-UI Radio Group (v0.1.5)](https://www.radix-ui.com/docs/primitives/components/radio-group) for details. This only covers **styple** abstractions.

#### Radio

A styled abstraction of `RadioGroup.Item` and `RadioGroup.Indicator`.

#### RadioGroup

A styled component wrapper for `RadioGroup.Root`.
