### API Reference

See [Radix-UI Tabs (v0.1.5)](https://www.radix-ui.com/docs/primitives/components/tabs) for details. This only covers **styple** abstractions.

#### Tabs

A styled component wrapper for `Tabs.Root`.

#### TabsList

A styled component wrapper for `Tabs.List`.

#### TabsTrigger

A styled component wrapper for `Tabs.Trigger`.

#### TabsContent

A styled component wrapper for `Tabs.Content`.
